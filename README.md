### 앞으로의 스터디 진행 방향

- **스터디 모임**의 경우 불참 일정 있을 시 날짜를 옮겨서 주 1회 무조건 모일 수 있도록 하기
- 스터디 시간에는 인당 문제설명 제한시간(5분) 두고 빠르게 코드리뷰
 
- BOJ 커리큘럼에 따라 한달 타이트하게 진행
- 9/9 : (2), (11), (12) 개념의 문제 중 플래티넘 제외하고 모두 풀기(골드 수준의 문제에서 못 풀 경우 풀이 봐오기)
- 강사님이 주시는 SWEA 문제는 각자 해결하기
